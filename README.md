# Fundamentos React
 
 Projeto para praticar e registrar os principais fundamentos do React na prática, com exemplos simples e navegação entre páginas.
 
 ## Tecnologias
 
 - **React**
 - **TypeScript**
 - **Vite**
 - **React Router DOM** (rotas)
 - **Material UI (MUI)** (componentes e layout)
 
 ## Como rodar o projeto
 
 - **Instalar dependências**
 
 ```bash
 npm install
 ```
 
 - **Rodar em modo desenvolvimento**
 
 ```bash
 npm run dev
 ```

 - **Preview do build**
 
 ```bash
 npm run preview
 ```

 ## O que tem no projeto
 
 - **Home** com botões para navegar para os exemplos.
 - Páginas com explicações em texto e pequenos exemplos interativos.
 - Componentes reutilizáveis (ex.: botão de navegação, voltar para Home).
